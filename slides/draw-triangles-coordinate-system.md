##  Draw Triangles - Coordinate System

* canvas要素(描画領域)の中心が (0, 0)
* 2つの値で1つの頂点の座標 (x, y)
* 3つの座標で1つの三角形
* 三角形が6つあります

```javascript
var vertexData = [
  -0.6,-0.4, -0.6,0.4, 0.6,-0.4,
  0.6,-0.4, 0.6,0.4, -0.6,0.4,
  -0.6,0.1, -0.7,0, -0.6,-0.1,
  0.6,0.1, 0.7,0, 0.6,-0.1,
  -0.4,-0.4, -0.3,-0.5, -0.2,-0.4,
  0.2,-0.4, 0.3,-0.5, 0.4,-0.4
];
```
